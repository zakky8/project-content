# Brand Guidelines

**Enforceable brand standard for the Astarter Web 4.0 / DeFAI positioning.** Aligned to the *Quiet Substrate* design philosophy. This document is the source of truth — any asset (internal or external) that contradicts these rules is off-brand.

**Version:** 1.0 · 2026-05-24
**Owner:** Astarter brand desk
**Audit cadence:** quarterly

---

## I. Voice & messaging

### Core positioning (one sentence — never alter)
> *Astarter is building the settlement layer for the autonomous AI Agent economy.*

### Approved expansion (two sentences — never alter without brand owner sign-off)
> *Astarter is building the settlement layer for the autonomous AI Agent economy. The Cardano-native infrastructure platform combines AI Agent, DePIN, and DeFAI primitives under a single coordinated stack.*

### Voice attributes (in priority order)

1. **Analytical** — observation precedes assertion; numbers precede opinions
2. **Restrained** — never hyperbolic, never marketing-jargon-dense, never EMOJI-stacked
3. **Comparative** — name peers explicitly to demonstrate analyst-grade thinking
4. **Forward-honest** — acknowledge what is not yet proven; bound the claim to the time horizon
5. **Discipline-coded** — language of infrastructure (substrate, primitive, rail, layer), not retail (moon, lambo, gem)

### Words to use freely
*infrastructure, substrate, primitive, settlement, rails, layer, stack, deterministic, economic actor, agent-native, financial primitive, execution layer, distribution layer, autonomous, coordinated, unit of account, payment medium, governance instrument, collateral primitive*

### Words to avoid in external copy
*revolutionary, game-changing, disrupt, ecosystem (overused), best-in-class, unparalleled, gem, moon, ape, degen, WAGMI, NFA, alpha (as noun), shillers, ngmi, breakthrough, paradigm shift (used 3+ times)*

### Tone differences by channel

| Channel | Tone | Example opener |
|---|---|---|
| Long-form editorial | Analytical thesis | *"The defining trade of the 2026 cycle is no longer about..."* |
| News flash | Factual lede | *"On [date], [actor] [verb]..."* |
| Twitter thread | Sharp hook | *"The defining trade of 2026 isn't [X]. It's [Y]."* |
| Discord / community | Direct + warm | *"@everyone — strategic update."* |
| LinkedIn (exec) | First-person | *"We started Astarter in 2021 to..."* |
| Investor / partner | Confident + precise | *"Three reasons settlement is the right slot..."* |
| Newsletter | Conversational | *"Hi, a short note on something we've been quietly building."* |

### Palette use rules

- **Substrate Deep** is the *only* approved primary background. Never white-background designs.
- **Cyan Signal** is exclusively for *one* element per composition: focal point, key data, or interactive accent. Multiple cyan elements dilute the signal.
- **Astarter Violet** is reserved for brand actions (CTA pills, primary buttons, links). Never used as decoration.
- Gradients use only Substrate Deep → Substrate Mid → Astarter Violet. No other color combinations approved.

## III. Typography

### Type stack (enforced)

| Weight | Use | Font | Fallback |
|---|---|---|---|
| Display / headline | Bahnschrift Bold | DIN-class, geometric, condensed authority | Inter / Helvetica Neue Bold |
| Subhead | Segoe UI Bold | Clean sans, broad readability | Helvetica Neue Bold |
| Body | Segoe UI | Default reading | Helvetica Neue / Arial |
| Spec-sheet / meta | Consolas | Monospaced, instrument feel | SF Mono / Roboto Mono |
| Chinese | Microsoft YaHei Bold / Regular | YaHei for digital, no decorative CN fonts | SimHei |

### Type scale

| Use | Size (digital) | Line height | Tracking |
|---|---|---|---|
| Hero headline | 42–88 pt | 1.05 | -0.02em |
| Section heading (H2) | 24–32 pt | 1.15 | -0.01em |
| Subhead (H3) | 16–20 pt | 1.25 | 0 |
| Body | 14–16 pt | 1.45 | 0 |
| Meta / spec | 9–11 pt | 1.2 | 0.05em (slight expansion) |
| Eyebrow / kicker | 10–12 pt | 1.0 | 0.08em |

### Typographic rules

- Headlines are **always** in DIN-class display weight, never script, never serif
- Caps used **only** for: kickers, spec labels, single-word punctuation (e.g., "ECONOMY.")
- Italic used **only** for: subtitles directly under titles, quoted source material
- **Never** mix more than 3 fonts in one composition
- **Never** use drop shadows, outline text, or gradient text fills

## IV. Logo usage

### Approved marks (assumes brand owner supplies official PNG/SVG)

- **Primary lockup** — full wordmark + icon (horizontal)
- **Compact lockup** — wordmark only (for narrow contexts)
- **Icon only** — for favicon, social avatar, app icon
- **Reversed** — white on dark (default for our palette)
- **Mono** — single-color version for monochrome contexts

### Clear space rule

Minimum clear space around the logo = the height of the cap-height of the wordmark in all four directions. Never crop closer.

### Minimum sizes

| Use | Min width |
|---|---|
| Web header | 120px |
| Mobile header | 80px |
| Favicon | 16px (icon-only) |
| Social avatar | 32px (icon-only) |
| Print | 0.5 inch |

### DO NOT

- Stretch or skew the logo
- Add drop shadow, outline, or glow effects
- Re-color the logo (only approved variants)
- Place over busy backgrounds without sufficient contrast
- Combine with other logos without proper lockup spacing
- Use AI-generated "A" placeholders in publication assets (only the real logo)

### Co-branding lockup (with UXLINK or other partners)

```
[ASTARTER LOGO]  ×  [PARTNER LOGO]
```

- Both logos sized to equal cap height
- Separator: `×` in Bahnschrift Regular, 60% opacity
- Equal padding around both
- Order: Astarter first when announcement is Astarter-led; partner first when partner-led

## V. Visual system (Quiet Substrate)

### Core elements (any new asset must include at least 3 of these)

- **Substrate Deep gradient background** (top-left dark → bottom-right violet)
- **Hex grid overlay** with focal alpha falloff toward the focal point
- **Isometric specimen** (cube, geometric solid) at focal area, with cyan core
- **Left-anchored editorial column** with structural vertical rule
- **Corner reticles** (cyan tick marks, 6–10 px arms)
- **Spec-sheet meta column** (right side, monospace labels + values)
- **Tick scale** along bottom edge (instrument feel)
- **Vignette** (subtle, never overpowering)

### Composition discipline

- Single focal point per composition — never two competing
- Left-edge content column = strict 60–80 px from edge
- Right-edge spec column = strict 140–180 px from edge
- Headline never exceeds 75% of canvas width
- CTA element placement: lower-left for hero, center for square / vertical
- Negative space: minimum 25% of canvas should be empty (this is non-negotiable for the *quiet* in Quiet Substrate)

### DO NOT

- Use stock photography
- Use generic 3D rendered abstract shapes (spheres, cones, organic blobs)
- Use crypto-cliché imagery (coins, charts, rocket emojis, lightning bolts)
- Use round podium / pedestal compositions
- Use Y2K aesthetics, vapor wave, or "tech bro" aesthetics
- Use multiple cyan elements in one composition
- Use color outside the approved palette

## VI. Asset audit checklist

Every new asset must pass these checks before publication:

- [ ] Background uses only approved gradient
- [ ] Palette restricted to the 8 approved colors
- [ ] Typography uses only Bahnschrift / Segoe UI / Consolas / YaHei
- [ ] Type sizes within the approved scale
- [ ] Logo placed with proper clear space and approved variant
- [ ] Headline within the 75% canvas width rule
- [ ] One cyan focal element only
- [ ] Negative space ≥25% of canvas
- [ ] Voice / wording within approved attributes
- [ ] No banned words (see § I)
- [ ] No banned visual elements (see § V)
- [ ] Mobile readability check passed if used in social
- [ ] Final review signed off by brand owner

## VII. Common violations (already seen in v1–v3 — fix going forward)

| Issue | Fix |
|---|---|
| AI-generated "A" mark in placeholder | Replace with real Astarter icon in Figma post-process |
| Headlines longer than 75% canvas width | Tighten copy or break into two lines |
| Multiple cyan accents in one composition | Reduce to one focal cyan; demote others to Lilac or White |
| Calibri / Arial in slide assets | Use Bahnschrift consistently |
| Emoji-heavy social copy | Limit to ≤3 emojis per post |
