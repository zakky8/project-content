# Part 1 — Glossary

**Use:** drop into any long-form piece for non-specialist readers; use as basis for landing-page tooltips; use to align internal team on shared vocabulary.

| Term | Plain-English definition | What it's NOT |
|---|---|---|
| **AI Agent** | An autonomous software entity that can perceive its environment, make decisions, and take actions without human input. Examples: trading bots, autonomous research agents, market-making algorithms. | Not the same as a chatbot. Chatbots respond to prompts; agents originate actions. |
| **Agent Economy** | The set of all economic activity — transactions, services, value exchange — initiated by autonomous agents rather than humans. | Not a future hypothesis. Already measurable on Polymarket, in DeFi automation, and via agent-driven trading. |
| **Autonomous** | Operating without per-decision human approval. The agent has been given goals and constraints; it chooses tactics. | Not "fully unsupervised." Most autonomous agents operate within designed boundaries. |
| **Cardano** | A proof-of-stake blockchain protocol founded by Charles Hoskinson (a co-founder of Ethereum). Uses an eUTXO transaction model. Astarter is Cardano-native. | Not a slower or older version of Ethereum. Architecturally different. |
| **Collateral primitive** | A standardized on-chain mechanism for posting assets as security against a loan or other financial obligation. | Not the same as a generic "deposit." Collateral primitives have specific over-collateralization, liquidation, and recovery rules. |
| **DeFAI** | The intersection of decentralized finance and autonomous AI execution. Financial primitives engineered for non-human counterparties. | Not "DeFi with AI features." It's DeFi *for* AI, structurally different. |
| **DeFi** (Decentralized Finance) | On-chain financial services that operate without traditional intermediaries. | Not just "crypto." DeFi specifically refers to permissionless financial primitives. |
| **DePIN** (Decentralized Physical Infrastructure Networks) | Networks that coordinate physical infrastructure (compute, storage, sensors, wireless) via on-chain incentives. | Not just any infrastructure-themed crypto project. DePIN specifically involves physical, real-world resources. |
| **DEX** (Decentralized Exchange) | An on-chain venue where assets are traded without a custodial intermediary. | Not the same as a CEX (centralized exchange). DEXs use smart contracts, not order books typically. |
| **Embargo** | An agreement that information may be received in advance but published only after a specified date / time. | Not the same as a press release "under wraps." Embargos require explicit acceptance. |
| **EMURGO** | One of the three founding entities of the Cardano protocol. The commercial arm of the Cardano ecosystem. Backed Astarter as a 2021 joint venture. | Not the same as Cardano Foundation or IOHK (the other two founding entities). |
| **ISPO** (Initial Stake Pool Offering) | A Cardano-native token distribution mechanism where users stake ADA to a project's stake pool and receive the project's token as reward (without losing their ADA). | Not the same as an ICO. Users don't send funds to the project. |
| **Launchpad / Launchpool** | A platform that supports the issuance and initial distribution of new tokens on a blockchain. | Not the same as an exchange listing. Launchpads precede exchange listing. |
| **Money Market** | A class of DeFi protocol providing collateralized lending, borrowing, and interest rate markets for crypto assets. | Not the same as a traditional money market fund. On-chain, permissionless, algorithmic. |
| **Settlement** | The process by which a transaction's value transfer is finalized — irreversible, recognized by all parties, recorded on-chain. | Not the same as "transaction submitted." Settlement is final confirmation. |
| **Settlement layer** | The infrastructure layer where value clearing between counterparties happens with deterministic finality. | Not the same as a payment layer (UX) or a coordination layer (matchmaking). Settlement is the bottom. |
| **Tech Service Platform** | Astarter's infrastructure tooling product for builders deploying on Cardano and adjacent ecosystems. | Not a standard term across DeFi. Specific to Astarter's stack. |
| **Token utility** | The set of functional roles a token plays within its protocol — governance, payment, collateral, fee discount, etc. | Not the same as token price. Utility describes function; price reflects market judgment of that function. |
| **UXLINK** | A Web3 social-growth network operating one of the larger active social graphs in crypto. Astarter's strategic alliance partner announced May 15, 2026. | Not a wallet, not a launchpad. Distribution-layer infrastructure. |
| **Web 4.0** | A framing of the next phase of internet infrastructure in which autonomous AI agents, rather than humans, originate the majority of transactions. | Not a marketing term invented by any one project. Used by Decrypt, KuCoin, multiple research analysts. |
| **AA token** | Astarter's native token. Fixed maximum supply of 100,000,000. Functions across the stack as unit of account, payment medium, governance, and collateral primitive. | Not the same as ADA (the native token of Cardano). |

# Part 2 — Anti-FUD Response Library

**Use:** community moderators, support team, anyone responding to challenges in Discord / TG / Twitter replies.

**Rules of engagement:**

1. **Never delete dissent** — it makes things worse. Engage or mute, never erase.
2. **Never argue beyond two exchanges** — if the conversation isn't converging, exit with grace.
3. **Never improvise on legal or regulatory topics** — escalate to comms + legal.
4. **Always link the source** — if your answer references the FAQ or thesis, link it.
5. **Match register** — calm, factual, slightly understated. Never defensive or aggressive.
