## How a real launch uses this package

A coordinated launch (per `29-embargo-multi-outlet-strategy.md`):

1. **T-7 days:** Internal all-hands using `30-internal-comms-package.md`
2. **T-72h:** Embargo offer to Tier-1 outlet (CoinDesk recommended)
3. **T-48h:** Embargo confirmed; full package shared
4. **T-24h:** Community holders informed (`31-community-holder-comms.md`)
5. **T-24h:** UXLINK co-marketing kit handoff (`32-uxlink-co-marketing-kit.md`)
6. **T-2h:** Tier-2 outlets warned of imminent drop
7. **T-0:** Embargo lifts; CoinDesk publishes
8. **T+5min:** Tier-2 broad drop (12+ outlets)
9. **T+1h:** X thread + LinkedIn exec posts begin
10. **T+2h:** Discord + Telegram + community channels live
11. **T+4h:** CN regional drop (Cointime CN + PANews + BlockBeats)
12. **T+12h:** KO + JP + ES regional drops (`34-multilingual-news-flash.md`)
13. **T+24h:** Reddit posts (`19-reddit-posts.md`)
14. **T+24h:** Day-1 metrics review using `measurement-framework.xlsx`
15. **T+7 days:** AMA / Twitter Spaces using `18-ama-script-spaces.md`
16. **T+7 days:** Day-7 retrospective using `37-post-launch-retrospective-template.md`
17. **T+30 days:** Day-30 full retrospective
18. **Phase 2:** Thought-leadership piece `35-thought-leadership-original-data.md` published as compounding asset

## Editorial discipline (per `C:\Users\rog\CLAUDE.md`)

- Primary sources only. Affiliate sites = starting points, never proof.
- Two-source minimum for any factual claim; single-source claims marked UNVERIFIED.
- Exact quotes for any rule, number, or threshold.
- Every fact has `accessed_at_utc` and a tier.
- Contradictions logged, never silently resolved.
