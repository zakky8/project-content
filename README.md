## Editorial discipline (per `C:\Users\rog\CLAUDE.md`)

- Primary sources only. Affiliate sites = starting points, never proof.
- Two-source minimum for any factual claim; single-source claims marked UNVERIFIED.
- Exact quotes for any rule, number, or threshold.
- Every fact has `accessed_at_utc` and a tier.
- Contradictions logged, never silently resolved.

## Visual system

All designed deliverables share one design philosophy — **Quiet Substrate** — applied across PDF, PPTX, and image variants. Palette: `#0F0820` deep, `#2D1B5F` mid-violet, `#6F00FF` violet, `#00E5FF` cyan signal. Typography: Bahnschrift (DIN-class) for display, Segoe UI for body, Consolas for spec-sheet meta.

**Asset workflow note:** every PNG has a clean reticle space top-left intended for compositing the real Astarter logo in Figma before publishing. AI generation cannot render brand-specific marks reliably and must not be trusted to do so.
