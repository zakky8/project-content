## 总体进度概览

| 状态 | 含义 |

| 🟢 已发布 | 文章已在该媒体上线 |
| 🟡 已发送 | 稿件已发给媒体编辑，待安排发布 |

## 媒体反馈日志

格式：日期 + 媒体 + 反馈内容 + 跟进动作

```
[日期] [媒体名]: [反馈内容]
跟进动作: [我们做了什么]
```

例：
```
2026-05-25 PANews: 编辑反馈希望增加 UXLINK 联盟更具体的产品集成时间
跟进动作: 已答复 — H2 2026 落地，具体时间随产品 Beta 公告同步
```

---

## 关键提醒

- **节奏控制：** 同一动态不要在同一周内发到 5 家以上媒体，避免内容稀释
- **质量优于数量：** 一家头部媒体的深度报道 > 五家小媒体的复制粘贴
- **反馈闭环：** 每发一篇都向飞哥确认是否上线 + 记录链接
- **周度复盘：** 每周五与 GK 同步本周分发情况 + 下周计划
