## Workflow for using this pack

1. **GK reviews** `03-cointime-followup-message.md` — send to Nicole as-is (CN version)
2. **GK + comms align** on which of the 12 dynamics get prioritized for next 7 days
3. **Cointime + 飞哥** pull from `05-promo-flashes-CN.md` per dynamic
4. **Visual asset** from `visuals/` is provided with each flash
5. **Outlets publish** — log in `press-pickup-tracker.xlsx`
6. **Cycle repeats** weekly with rotating dynamic focus

## Editorial discipline (same as `astarter-cointime-2026-05-24/`)

- Primary-source only; affiliate sites = starting points, never proof
- Two-source minimum for any factual claim; single-source = UNVERIFIED
- Exact quotes for any rule, number, or threshold
- No fabricated dates, numbers, partnerships, or quotes
- All forward-looking statements clearly forward-looking
